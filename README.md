 - left/right click on the iface name allow you to change view across different interface

## How to:

To start the daemon and open all the widgets

> eww daemon; eww open sys-info; eww open cpu-usage; eww open mem-usage; eww open net-usage; eww open kalendar

To close all the widget and terminate the daemon

> eww close-all; eww kill
